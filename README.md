# Rachel Capital

个人投资委员会项目，用于持续记录账户状态、投资原则、组合复盘、重大事件与决策日志。

## 项目目标

以组合管理而非个股预测为核心，持续优化投资决策，目标是在控制风险的前提下修复净值并实现长期增长。

## 当前阶段

**Phase 2：净值修复与组合优化**

- 初始资金参考：60 万
- 当前资产基准：41.9 万
- 当前现金：9 万
- 当前组合健康度：83 / 100

## 使用方式

每周查看 `Dashboard.md`，每次交易记录到 `Decisions.md`，每周复盘写入 `Weekly/`，每月战略复盘写入 `Monthly/`，每只核心持仓的投资逻辑写入 `Thesis/`。

## 核心原则

1. 不预测短期涨跌，只管理组合。
2. 优先提升组合质量，而非追求快速回本。
3. 现金只用于高胜率机会。
4. 每周复盘，每月战略复盘，重大事件召开临时投资委员会。
