# Templates

This directory stores reusable templates for Rachel Capital.

Templates are source documents for new operating records. They are not the live operating state.

## Usage Rules

- `Dashboard.md` at the repository root is always the live dashboard.
- New weekly records should be created from `templates/Weekly_Review_Template.md`.
- New monthly records should be created from `templates/Monthly_Review_Template.md`.
- New committee meeting records should be created from `templates/Meeting_Template.md`.
- Template files should remain generic and reusable.
- Current state should never be written into a template file.
