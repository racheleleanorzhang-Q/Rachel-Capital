# Public Funds

This file tracks public funds first, and uses insurers and foreign capital as supporting observation layers.

## Why Track It

This module helps detect:

- crowded trades versus under-owned names
- sector rotation pressure
- whether defensive or growth capital is leading
- whether long-term capital and short-term capital are moving in the same direction

## Coverage Scope

- public funds
- insurers
- foreign capital

These are supporting evidence sources, not primary authorities.

## Interpretation Rules

- Public fund behavior can be informative, but also more crowded and more benchmark-driven.
- Insurer behavior matters most for defensive, yield, and long-duration allocation signals.
- Foreign capital matters when it confirms or contradicts local institutional direction.
- Use this file to improve signal triangulation, not to chase popularity.
